## The Keys

### Description
- **Quantity:** 2
- **Material:** Brass
- **Size:** Small, ornate
- **Markings:** Numbered (1 and 2)
- **Origin:** From Mussolini's jacket

### Function
- Both must be inserted simultaneously
- Both must be turned at the same moment
- Disables booby-trap, allows safe opening
- Without both: chest explodes if forced

### Journey
1. Mussolini's jacket pocket → stolen by Paul
2. Paul → Jens (deathbed transfer)
3. Jens → suitcase in crawl space (1952)
4. Suitcase → Ole (2019)
5. Ole → hidden in Porsche, then used to open chest
